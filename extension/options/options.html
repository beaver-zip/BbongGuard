BbongGuard

YouTube 가짜뉴스 팩트체커

API 설정

확장프로그램이 정상적으로 작동하려면 YouTube Data API v3 키가 필요합니다.

Google Cloud Console에서 발급받을 수 있습니다.

로컬 추론 서버의 주소를 입력하세요. (기본값: http://localhost:8000)

API 키 발급 방법

  1. Google Cloud Console에 접속합니다
  2. 새 프로젝트를 생성하거나 기존 프로젝트를 선택합니다
  3. "API 및 서비스" > "라이브러리"로 이동합니다
  4. "YouTube Data API v3"를 검색하고 사용 설정합니다
  5. "사용자 인증 정보" > "사용자 인증 정보 만들기" > "API 키"를 선택합니다
  6. 생성된 API 키를 복사하여 위 입력란에 붙여넣습니다

API 할당량 안내

YouTube Data API v3는 일일 10,000 units의 무료 할당량을 제공합니다.

  • videos.list: 1 unit (영상 정보)
  • commentThreads.list: 1 unit (댓글 20개)
  • search.list: 100 units (관련 영상 9개)

영상 1개 분석 시 약 102 units가 소모되어, 하루에 약 98개의 영상을 분석할 수 있습니다.
